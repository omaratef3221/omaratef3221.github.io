## How to Update Content

### Manual Updates (About, Experience, Skills)
Simply edit the JSON files in the `data/` folder:

**data/about.json** - Update your bio, education, location
**data/experience.json** - Add/edit work experience entries
**data/skills.json** - Update skills and proficiency levels

### Automatic Updates (Publications, GitHub)
Run the Python update script to fetch the latest data:

```bash
python update_data.py
```

This will:
1. âœ… Fetch latest publications from Google Scholar
2. âœ… Get citation counts and h-index
3. âœ… Fetch your top GitHub repositories
4. âœ… Update `data/scholar.json` and `data/github.json`

**Requirements:**
```bash
pip install scholarly requests
```

**Note:** The files `experience.json` and `about.json` are **NOT** updated by the script - you update these manually as needed.

Edit CSS variables in `styles.css`:
```css
:root {
    --background-dark: #000a0f;
    --background-darker: #000508;
    --secondary-color: #00d4ff;
    --accent-color: #00ff88;
    --text-primary: #ffffff;
```

### Contact Form
The contact form uses Formspree. To set it up:
1. Sign up at [formspree.io](https://formspree.io) (free)
2. Create a new form
3. Replace the form action in `index.html` (line 619):
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

### Publications Per Page
Change the constant in `scholar-integration.js` (line 9):
```javascript
const PUBLICATIONS_PER_PAGE = 3;  // Change to desired number
```

### Skills Display
Modify the pill design in `load-skills.js` or adjust styles in `styles.css` under `.skill-pill`.
