# Company Logos

Place company logos in this directory as PNG or JPG files.

## Expected Files:
- `almosafer.png` - Almosafer logo
- `virtue.png` - Virtue Therapy logo
- `uos.png` - University of Sharjah logo

## Logo Requirements:
- Format: PNG or JPG (PNG with transparent background recommended)
- Size: Square aspect ratio (e.g., 200x200px, 300x300px)
- Background: Transparent or white
- File size: Keep under 100KB for fast loading

## How to Add Logos:
1. Save your company logo in this folder
2. The logo will automatically appear in the experience cards
3. If a logo file is not found, the section will gracefully hide the logo space

## Optional:
Logos are optional! If you don't provide a logo file, the experience cards will still display correctly without it.
